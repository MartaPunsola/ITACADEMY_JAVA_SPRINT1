Instructions:

1. Open your computer's terminal (CMD).

2. Write "cd" followed by the path of the directory where your file is located. 
For example: cd "C:\Users\Marta Punsola\eclipse-workspace\TascaS1_05\src"

3. First, you will need to compile the files you want to execute. Write "javac" followed by the name of the package and the name and extension of the file.
For example: javac n1exercici5/Product.java n1exercici5/Main_n1exercici5.java

4. Then, execute the file writing "java" followed by the name of the package, the name of the file (without the extension) and the name of the package.
For example: java n1exercici5/Main_n1exercici5 n1exercici5

The program will proceed to serialize and deserialize an object.
